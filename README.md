# 📊 Sorting Algorithm Visualizer

An interactive web-based **Sorting Algorithm Visualizer** that demonstrates how different sorting algorithms work in real-time through dynamic animations.

This project helps users understand sorting techniques visually by showing step-by-step transformations of data.

---

## 🚀 Features

### 🎯 Real-Time Visualization
- Visual representation of sorting processes  
- Step-by-step animation of array transformations  
- Color-coded comparisons and swaps  

---

### ⚡ Multiple Sorting Algorithms
- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Merge Sort  
- Quick Sort  

---

### 🎛️ User Controls
- Adjustable array size  
- Speed control for visualization  
- Generate new random arrays  
- Start, pause, and reset functionality  

---

### 🌈 Interactive UI
- Clean and responsive interface  
- Smooth animations for better understanding  
- Designed for both beginners and advanced learners  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript / React  
- **Visualization:** DOM manipulation / Canvas  
- **Styling:** CSS / Tailwind CSS  

---

## ⚙️ How It Works

1. User generates a random array  
2. Selects a sorting algorithm  
3. Visualization starts showing comparisons and swaps  
4. Bars rearrange until the array is sorted  

---

## 🎯 Learning Outcomes

- Understand how sorting algorithms operate internally  
- Compare time complexity visually

## 📈 Algorithms Complexity

| Algorithm       | Best Case | Average Case | Worst Case |
|----------------|----------|-------------|------------|
| Bubble Sort    | O(n)     | O(n²)       | O(n²)      |
| Selection Sort | O(n²)    | O(n²)       | O(n²)      |
| Insertion Sort | O(n)     | O(n²)       | O(n²)      |
| Merge Sort     | O(n log n) | O(n log n) | O(n log n) |
| Quick Sort     | O(n log n) | O(n log n) | O(n²)      |
